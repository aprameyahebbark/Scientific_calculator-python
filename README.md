# Advanced Scientific Calculator

A Python-based scientific calculator that supports advanced mathematical operations, calculation history, memory functions, and robust error handling.

## Features

- Scientific calculations
- Calculation history
- Memory recall functionality
- Input validation and error handling
- Modular Python code structure
- Unit tests included

## Project Structure

```
.
├── calculator.py
├── history.py
├── main.py
├── utils.py
├── requirements.txt
├── tests/
└── screenshots/
```

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
